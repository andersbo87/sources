Man kan bruke JS til å aktivere og deaktivere klasser i CSS.

I steden for å gjøre ting manuelt, slik…:
// SELECT
var tag = document.getElementById("highlight");

// MANIPULATE
tag.style.color = "blue";
tag.style.border = "10px solid red";
tag.style.fontSize = "70px"
tag.style.background = "yellow";
tag.style.marginTop = "200px"

… kan man bruke klasser isteden, slik:
// I CSS-fila:
.klassenavn {
  color: blue;
  border: 10px solid red;
  fontSize: 70px;
  background: yellow;
  marginTop: 200px
}

// I JavaScript:
var tag = document.getElementById("highlight");
tag.classList.add("klassenavn");
eller:
tag.classList.toggle("klassenavn"); // denne metoden oppretter klassen om den ikke finnes. Finnes den allerede, blir klassen fjernet.

Man kan hente tekstinnhold i en HTML-tag på to måter (JS):
1) vha textContent:
     var p = document.querySelector("p");
     p.textContent;
   eller
2) vha innterHTML:
     var p = document.querySelector("p");
     p.innerHTML;

Skal man endre teksten og bruke en HTML-tag inne i teksten, må man bruke
innerHTML fordi textContent ikke behandler tekst som HTML.
p.innerHTML = "Hei <strong>på</strong> deg!"; erstatter den gamle teksten med
"Hei på deg" hvor på er i fet skrift.
p.textContent = "Hei <strong>på</strong> deg!" erstatter den gamle teksten med "Hei <strong>på</strong> deg!" (HTML-taggen skrives som om den var en del av teksten.)

Vi har en <a>-tag og en <img>-tag:
<!-- HTML -->
<a href="www.google.com">Dette er ei lenke</a>
<img src="logo.png">

var lenke = document.querySelector("a"); // kunne også brukt document.getElementsByTagName("a")[0];
